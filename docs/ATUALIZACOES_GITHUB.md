## 🎯 **Como usar agora:**

### **1. Executar o App**
```powershell
# Setup automático
.\scripts\Setup-And-Run-AurantisSync.ps1

# Ou manual
python aurantis_sync_mvp.py
```

### **2. Criar Executável**
```powershell
# PowerShell (Recomendado)
.\scripts\Create-Executable.ps1

# Python
python build_exe.py

# Batch
create_exe.bat
```

### **3. Testar Estrutura**
```powershell
.\scripts\test_structure.ps1
```

### **4. Upload para GitHub**
```bash
python upload_github.py
```
